Spartan Explorer

This is the website with which you can learn all there is to do on Michigan State's Fine Campus. It is currently under development, but this is a rough outline of the website's layout.

Categories

Each category currently has its own page where one can view each event, a short description with a picture, and access a comments board to see what other users think of the event or location.

The comments board and rating system have not yet been implemented, and will be as the training is completed in related fields.

Currently, only 3 items in each category have been posted. As further research goes in to finding more of these events and locations, they will be added, it is also the goal that there will be a function for users to suggest an item to the lists and after being verified for accuracy and to avoid duplicates, it can be added to the website.

Map

A central part of the website is the map system. A javascript code for creating maps called Leaflet is currently being observed with hopes to implement a fully-functional map so as to provide a visual location to each of these elements.

The current map is simply a picture file so as to provide an idea. When the page is zoomed in on, the header shrinks in several increments. In the final page, the header will be stationary and the map will have its own zoom controls.

The map will have access to and provide the position of all of the events and locations listed on other pages.

Design

Spartan Explorer was designed with simplicity in mind. Each page is layed out in a predictable, reliable way. Navagation controls always remain visible so users can jump between pages with ease. Each page follows a simple and easy to understand format, while providing the needed information.

Spartan Explorer follows the official Spartan Branding color scheme and provides many beautiful pictures of Michigan State's campus to allow any spartan user to feel right at home.

There are still many more things that need to be done before Spartan Explorer can achieve its full potential, but it is off to a good start.
